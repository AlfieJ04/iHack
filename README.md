# Hack-Attack

AlfieJ04's Hack-Attack Application (Work in progress)

A graphical user interface for running popular hacking automation scripts.
Designed for use with Kali Linux.

# Install dependencies

sudo apt-get install dpkg-dev build-essential python3.7 python3.7-dev python3-pip libpython3.7-dev freeglut3-dev libgl1-mesa-dev libglu1-mesa-dev libgstreamer-plugins-base1.0-dev libgtk-3-dev libjpeg-dev libnotify-dev libpng-dev libsdl2-dev libsm-dev libtiff-dev libwebkit2gtk-4.0-dev libxtst-dev

# Clone repository

git clone https://github.com/AlfieJ04/Hack-Attack.git

cd Hack-Attack

# Install pip dependencies

pip install -r requirements.txt
